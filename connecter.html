Bonne question ! 💡 Renommer un fichier dépend de l'outil ou système que tu utilises. Voici comment faire selon ton cas :

📁 Si tu es sur Windows :

  1. Clique droit sur le fichier.

  2. Clique sur Renommer.

  3. Tape le nouveau nom (ex. preventes.html) puis appuie sur Entrée.

🍏 Si tu es sur Mac :

  1. Clique une fois sur le fichier pour le sélectionner.

  2. Appuie sur la touche Entrée/Retour.

  3. Le nom devient modifiable. Tape le nouveau nom (ex. index.html) et appuie sur Entrée.

💻 Si tu utilises Visual Studio Code (VS Code) :

  1. Clic droit sur le nom du fichier dans la barre latérale.

  2. Choisis Rename ou Renommer.

  3. Écris le nouveau nom du fichier (n'oublie pas l'extension .html) et appuie sur Entrée.

⚠️ Astuces importantes :

  • Toujours mettre .html à la fin si c’est un fichier HTML.

  • Évite les espaces dans les noms de fichiers. Utilise plutôt des tirets (-) ou des underscores (_), par exemple : conseils-utilisation.html
